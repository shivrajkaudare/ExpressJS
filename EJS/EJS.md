
<h2> Tags in EJS </h2>

1) <%    'Scriptlet' tag, for control-flow, no output                                         <br>
2) <%_   ‘Whitespace Slurping’ Scriptlet tag, strips all whitespace before it                 <br>
3) <%=   Outputs the value into the template (HTML escaped)                                   <br>
4) <%-   Outputs the unescaped value into the template                                        <br>
5) <%#   Comment tag, no execution, no output                                                 <br>
6) <%%   Outputs a literal '<%'                                                               <br>
7) %>    Plain ending tag                                                                     <br>
8) -%>   Trim-mode ('newline slurp') tag, trims following newline                             <br>
9) _%>  ‘Whitespace Slurping’ ending tag, removes all whitespace after it                     <br>
